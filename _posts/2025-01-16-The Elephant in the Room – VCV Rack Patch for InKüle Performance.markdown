The Elephant in the Room is a VCV Rack patch prepared for a 20-minute InKüLe performance shown as part of the UdK 50-year anniversary program on February 8, 2025.

## Patch setup

The patch generates ambient textures, chord swarms, and rhythmic layers. I use a **Korg NanoKontrol2** for live control:

- **Mixer channels**
  1. Drone
  2. Chord swarm
  3. Sequence pluck
  4. Drum part
  5. High texture layer
- **Rotary knobs**
  - Bernoulli gate probability
  - Delay modulation
  - Basic filter shaping

Patch link: [The Elephant in the Room on Patchstorage](https://patchstorage.com/the-elephant-in-the-room/)
